Binary sketch size: 3524 bytes (of a 4096 byte maximum)

Date Sat 01 September 2012 Tags cuffelinks / rag

Right I fitted the following in to less than 4096 bytes. https://github.com/bmsleight/cuffelink/blob/master/software/micro/arduino/tiny_rag.ino

  • Traffic Light Sequence, (Red, Red-Amber, Green, Amber, Red)
  • Pelican Sequence, (Red, Flashing Amber, Flashing Green, Green, Amber, Red)
  • Displayed Time (using Red = 1, Amber = 5, Green = 10)
  • Displayed Date (using Red = 1, Amber = 5, Green = 10)
  • Set Time
  • Set Date

Only ~500 bytes to spare.
