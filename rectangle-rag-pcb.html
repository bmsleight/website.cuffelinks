Rectangle RAG PCB

Date Wed 22 April 2015 Tags cuffelinks / traffic / PCB

Following some fun looking at outlines for laser-cut-traffic-light-cufflink.html I whipped up a quick PCB using KiCad. Was quite surprised when all the components fit on the PCB and the routing was possible, I thought it would not work.

You can view all the raw KiCad files in the git repository. I test the gerber files at oshpark although I have not ordered them.
